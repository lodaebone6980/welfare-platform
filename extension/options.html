GovMate Indexer - 설정

로컬 개발 시 http://localhost:3000 도 가능 (manifest 의 host_permissions 에 포함됨)
서버 .env 의 INDEXING_QUEUE_SECRET 값과 정확히 일치해야 합니다.
설치 체크리스트
  1. 서버 .env 에 INDEXING_QUEUE_SECRET="긴_랜덤_문자열" 추가
  2. 위 시크릿을 옵션에도 동일 입력
  3. 네이버 검색어드바이저(govmate.co.kr 사이트 선택), 다음 페이지등록 페이지에 미리 로그인
  4. 활성 체크 후 저장 → 5분 뒤 자동 시작
