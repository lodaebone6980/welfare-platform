# 지원금

## 찾으면 있고, 모르면 못 받는 돈

"지원금"은 정부·지자체·공공기관이 **특정 목적을 가진 국민에게 현금 또는 준현금**으로 지급하는 제도 전체를 뜻해요. 창업을 준비하는 청년에게, 자녀를 낳은 부부에게, 일자리를 잃은 근로자에게 각각 다른 형태로 지급됩니다.

문제는 제도가 너무 많고 복잡해서 **"내가 대상인지조차 몰라서 못 받는"** 경우가 많다는 점이에요. 고브메이트는 현재 {count}개의 지원금을 대상·테마별로 분류하고, 복잡한 조건을 한 번에 검토할 수 있게 해드립니다.

## 대상별로 찾는 지원금

### 청년 (만 19~34세)
- 청년도약계좌 / 청년내일저축계좌 (자산형성)
- 청년 월세 특별지원 / LH 청년매입임대
- 국민취업지원제도 / 내일배움카드
- 청년창업사관학교 / K-Startup 창업지원금

### 신혼부부·출산
- 출산장려금 (지자체별 100만~1천만)
- 첫만남이용권 200만
- 신혼부부 전세자금 대출 이자지원
- 아이돌봄·산모신생아 서비스

### 소상공인·자영업자
- 소상공인 정책자금 (직접대출·대리대출)
- 새출발기금 (채무조정)
- 노란우산공제 가입 지원
- 전기요금·공과금 감면

### 고령자·장애인·저소득
- 기초연금 / 장애인연금
- 노인일자리 사업
- 긴급복지지원제도 (생계·의료·주거)

## 지원금이 왜 누락되나요?

1. **신청주의** — 대부분 지원금은 신청해야 받을 수 있어요. 자동 지급은 예외입니다.
2. **조건의 미묘한 차이** — 중위소득 50%·80%·120%가 제도마다 다르고, 가구원 수 계산도 달라요.
3. **기간 제한** — 선착순, 예산 소진, 분기별 모집 등으로 1년 내내 상시 접수가 아닌 경우가 많습니다.
4. **증빙 서류** — 건강보험 자격득실확인서, 가족관계증명서 등 사전 준비가 필요합니다.

## 신청 플로우 (일반적)

1. **자격 자가진단** — 복지로 / 고브메이트 조건 확인
2. **서류 준비** — 신분증, 통장 사본, 소득·재산 증빙
3. **온라인 신청** — 정부24 / 복지로 / 각 부처 홈페이지
4. **자격 심사 (2~4주)** — 필요 시 대면 상담
5. **결정 통보** — 문자·우편·앱 알림
6. **지급** — 계좌입금 / 바우처 카드 / 현물

## FAQ

**Q. 여러 지원금을 동시에 받을 수 있나요?**
대부분 동시 수급이 가능하지만, **"중복 수급 금지"** 가 명시된 경우가 있어요. 예를 들어 청년 월세 특별지원과 주거급여 중 하나만 선택해야 하는 식입니다. 각 제도 상세페이지에서 "중복 불가 제도"를 꼭 확인하세요.

**Q. 작년에 신청해서 탈락했는데 다시 신청할 수 있나요?**
소득·자산·나이 등 **조건이 변했다면** 다시 신청 가능합니다. 일부 지원금은 "최초 1회 한정"이지만, 청년 지원금 중 다수는 회차별 재신청이 열려 있어요.

**Q. 지원금 "대리 신청" 대행업체가 있던데 써도 되나요?**
정부 지원금은 **본인이 직접 신청하는 것이 원칙**입니다. 수수료를 요구하는 대행업체는 대부분 불필요하거나, 최악의 경우 개인정보 유출·보이스피싱 위험이 있습니다.
