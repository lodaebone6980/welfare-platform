# 문의하기

고브메이트를 이용해 주셔서 감사합니다. 아래 안내를 참고해 주시면 더 빠르고 정확하게 답변드릴 수 있습니다.

## 자주 묻는 질문부터 확인해 보세요

문의 전 아래 내용을 먼저 확인해 보시면 대부분 해결됩니다.

### Q. 내가 받을 수 있는 지원금을 어떻게 찾나요?
홈 화면의 **검색창에 "월세, 청년, 출산"** 등 키워드를 입력하시거나, **카테고리 메뉴**(환급금/바우처/지원금 등)에서 조건별로 필터링할 수 있습니다.

### Q. 특정 제도 신청은 고브메이트에서 직접 하나요?
아니요. 고브메이트는 **정보 제공** 플랫폼입니다. 각 제도의 상세 페이지 하단 **"공식 신청 바로가기"** 버튼으로 담당 공공기관 사이트로 연결해 드립니다. 최종 신청은 공식 사이트에서 진행해 주세요.

### Q. 제도 정보가 틀렸거나 오래됐어요.
정말 죄송합니다. 아래 **오류 제보** 양식으로 알려 주시면 가능한 한 빠르게 확인·수정하겠습니다. 제도명·페이지 URL·잘못된 내용을 함께 보내 주시면 큰 도움이 됩니다.

### Q. 로그인이 안 돼요 / 비밀번호를 잊었어요.
로그인 페이지의 **"비밀번호 찾기"** 기능을 이용하세요. 해결되지 않는 경우 아래 양식으로 문의해 주세요.

### Q. 회원 탈퇴는 어디서 하나요?
마이페이지 → 계정 설정 → 탈퇴에서 가능합니다. 탈퇴 시 개인정보는 즉시 파기되며, 북마크·알림 설정도 함께 삭제됩니다.

## 문의 유형별 안내

### 🐛 서비스 오류 / 버그 제보
- 오류가 발생한 페이지 URL
- 사용 중인 기기·브라우저 (예: iPhone 15 / Safari)
- 발생 시각과 증상
- 가능하다면 스크린샷

### 📝 제도 정보 오류 제보
- 해당 제도명 또는 상세 페이지 URL
- 잘못된 부분
- 올바른 정보의 출처 (공공기관 링크)

### 🤝 제휴·협업 제안
- 단체명·기관명
- 담당자 성함·연락처
- 제안 개요

### 📣 서비스 개선 제안
- 개선이 필요한 부분
- 원하시는 방향

### ⚖️ 개인정보 관련 문의
- 개인정보 열람·정정·삭제 요청
- 연락 가능한 이메일

## 문의 양식

```
[이름]
[이메일]
[유형]  (오류 / 정보 정정 / 제휴 / 건의 / 기타)
[제목]
[내용]
```

※ 문의 페이지에 폼이 연결되어 있지 않은 경우 임시로 아래 이메일로 보내주실 수 있어요.

- 일반 문의: **hello@govmate.co.kr**
- 정보 정정 제보: **report@govmate.co.kr**
- 개인정보 관련: **privacy@govmate.co.kr**
- 제휴·제안: **partnership@govmate.co.kr**

*(도메인은 예시이며, 실제 운영 이메일은 서비스 내 공지에 따릅니다.)*

## 답변 안내

- 영업일 기준 **24~72시간** 내에 답변드립니다.
- 주말·공휴일은 답변이 지연될 수 있습니다.
- 제도 정보 정정은 확인 후 반영까지 **최대 7일**까지 걸릴 수 있습니다.

## 고브메이트는 이런 문의에는 답변드리기 어렵습니다

- 특정 지원금의 **개별 심사 결과**에 대한 문의 (해당 기관 담당 부서로 문의 부탁드립니다)
- **대리 신청**, 신청 서류 대행 요청
- 대출·투자·세무 등 **전문 상담** (해당 전문가·공식 기관 이용)
- 고브메이트에 등록되지 않은 **사적 거래**에 관한 분쟁
